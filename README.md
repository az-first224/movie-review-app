# 作品レビュー記録アプリ

映画、ドラマ、アニメ、ドキュメンタリーなどのレビューをブラウザ上で記録できるローカルWebアプリです。

## ファイル構成

- `index.html`: 画面のHTML
- `styles.css`: デザインとレスポンシブ対応
- `app.js`: 登録、編集、削除、検索、保存処理
- `README.md`: この説明書

## 使い方

1. `index.html` をブラウザで開きます。
2. 入力フォームに作品情報を入力します。
3. 星評価はスライダーまたは数値入力で、0.0から5.0まで0.1単位で指定できます。
4. カテゴリータグは `映画`、`漫画`、`ドラマ`、`アニメ`、`ゲーム`、`小説`、`音楽` から1つ選択します。
5. ジャンルは複数選択できます。
6. `登録する` を押すとレビュー一覧に追加されます。

## 主な機能

- 作品レビューの新規登録
- 登録済みレビューの一覧表示
- レビューの編集
- レビューの削除
- 作品タイトル、監督名、出演者、ジャンル、カテゴリータグ、コメント本文で検索
- 星評価で絞り込み
- ジャンルで絞り込み
- カテゴリータグで絞り込み
- 登録日時と更新日時の表示
- `localStorage` によるブラウザ内保存

## 動作確認方法

1. `index.html` を開きます。
2. 最初の起動時はサンプルデータが3件表示されます。
3. 新しいレビューを登録し、ブラウザを閉じてからもう一度開きます。
4. 登録したレビューが残っていれば保存は正常です。
5. キーワード検索、星評価、ジャンル、カテゴリーの各絞り込みを試します。
6. 任意のレビューで `編集`、`削除` が動くことを確認します。

## データ保存について

データは外部サーバーには送信されず、ブラウザの `localStorage` に保存されます。
同じPCでも別のブラウザで開いた場合は、保存データは共有されません。

保存データを初期化したい場合は、ブラウザの開発者ツールから Local Storage の `videoReviewLog.records` を削除してください。
